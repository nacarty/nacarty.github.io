Some Recent Projects


This is an application that interfaces with TWITCH TV API to look up specified channels and display the logo associated with the stream, the official name, the game being streamed, if any, the status of the channel and the number of followers, if relevant.

This is an application that looks up and displays the local weather of the current user. The application uses the browser's in-built function or ip-api.com to get the current geographic location. It then uses that information to look up weather data relevant to the location from Forecast.io

This is my own implementation of the well-known game, SIMON. However, I have added two twists: the colors shuffle and the player must in one case follow the colors, and in the other case, follow the quadrants and ignore colors. These twists add greater challenge and deeper interest.

This is my personal implementation of TIC-TAC-TOE using a minimax algorithm. The user has the option of using a board size greater than 3X3 up to 8X8. When the board size is greater than 3X3, the algorithm uses a depth-limited form of minimax because of the complexity of the algorithm.

This is my beautiful version of the POMODORO clock. This implementation required a healthy appreciation of Trigonometry and SVG graphics in HTML5. The user has the option of changing the display size of the clock.

This is a Javascript calculator that I intend to do further work on to make it a professional-grade application. This implementation allows the user to see the history of the calculations. It provides a number of fairly complex functions usually found on good calculators.

This application interfaces with wikipedia through wikipedia's API. It looks up lists of wikipedia documents and allows the user to opt to view any of those files. Conveniently, the user can choose any language as the basis of his/her search. There are over 200 languages provided for.

This application is called a random quote machine. It generates famous quotes by well-kown personalities like ALbert Einstein and others. There is a small database of sorts (a JSON file) fron which the quotes are randomly retrieved.

This is a thumbnail of IBM's Big Data University webpage where I have so far completed one course in the basics of Big Data. I am thinking of doing the course on Hadoop and a few others.


This is a console Poker application written in C# using Xamarin IDE. The game takes between 2 and 6 players. It deals as many hands as is possible and shuffles before dealing each hand.

This is a thumbnail of the webpage you are now looking at. It is simple website with some personal profile information, my recent projects and contact information.

This is my first freecodecamp.com project. Its is a simple website documenting some accomplishments of Hillary Rodham Clinton, first female nominee for President of the USA.
